Google Analytics

What is Google Analytics?

Google Analytics is a free web analytics tool offered by Google, that help you analyze your website traffic.

But how does Google Analytics Work?

  1. A JavaScript page tag is inserted into the code of each page.
  2. This tag runs in the web browser of each visitor, collecting data and sending it to one of Google's data collection servers.
  3. Then Google Analytics can generate customizable tracking reports and visualize data such as the number of users, bounce rates, average session durations, sessions by channel, page views, goal completions and other more.

However, because it uses cookies, the system can't collect data for users who have disabled them.

Google Analytics features includes:
  • data visualization and monitoring tools, including dashboards, scorecards and motion charts;
  • data filtering, manipulation and funnel analysis;
  • data collection application program interfaces (APIs);
  • predictive analytics;
  • segmentation for analysis of conversions;
  • custom reports for advertising, audience behavior and conversion;
  • email-based sharing and communication;
  • integration with other products, including Google Ads, Google Data Studio, Google Search Ads and etc.

Google Analytics aggregates the data collected from website in multiple ways by four levels:

  1. User level
  2. Session level
  3. Pageview level
  4. Event level

The dashboard can be viewed on the Google Analytics site and is available through a widget or a plugin for integration into other sites.

What is the most mportant metrics?

A metric is a standard of quantitative dimension. Google Analytics enables users to track up to 200 different metrics to score performing of their websites. These are the most popular metrics:

  • Users
  • Bounce rate
  • Sessions
  • Average session duration
  • Percent of new sessions
  • Pages per session
  • Goal completions
  • Page views

Google Analytics 4

Features that are new in Google Analytics 4 include:

  • machine learning and artificial intelligence (AI) tools;
  • deeper integration with Google Ads;
  • customer-centric reporting designed around lifecycle data;
  • additional codeless tracking features that can provide data with less delay;
  • enhanced data control features for regulatory compliance and data management.
